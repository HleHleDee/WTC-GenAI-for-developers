# WTC GenAI for Software Engineers

This repository contains my exercises from the WeThinkCode_ GenAI for Developers curriculum.

The main focus of this work is learning how to use AI as a tool to support software development, not just for coding but also for understanding, analysing, and improving code.


# What I Practiced

* Using AI to understand unfamiliar codebases
* Breaking down project structure and identifying entry points
* Analysing and improving code quality
* Learning programming concepts using structured prompts
* Comparing different frameworks such as FastAPI, Flask, and Django
* Practicing FastAPI fundamentals and API development
* Understanding advanced concepts like dependency injection and middleware
* Learning how to approach a new programming language using AI
* Writing reflections and documenting learning


# Key Areas Covered

* Python (Task Manager CLI project analysis)
* JavaScript (language features and code improvements)
* FastAPI (API development and architecture)
* AI Prompting Techniques for Developers


# Purpose of This Repository

This repository shows how I used AI to:

* Improve my understanding of programming concepts
* Learn new technologies step by step
* Analyse and work with real codebases
* Develop better problem-solving and thinking skills


# Notes

Most of the work is written in a learning tone as I am still developing my skills. The goal was to show my thinking process, not just final answers.
