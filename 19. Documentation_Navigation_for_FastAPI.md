-Makes applications easier to maintain


# Concept to Code Translation

**Concept**

Dependency Injection

**Simple Example**

```python
from fastapi import FastAPI, Depends

app = FastAPI()

def get_username():
    return "Hleziphi"

@app.get("/")
def read_root(user: str = Depends(get_username)):
    return {"message": f"Hello {user}"}
```

**My Understanding**

-The function get_username is reused
-Depends() injects it into the route
-This avoids repeating the same logic

    posts.append(title)
    return {"message": "Post created", "posts": posts}
